プライバシーポリシー

最終更新日：2026年5月8日

ごめんネ（以下「当サービス」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。 本ポリシーでは、収集する情報・利用目的・第三者提供・データの削除について説明します。

1. 収集する情報

アカウント情報

情報 説明
メールアドレス 登録・ログイン・メール認証に使用します。Amazon Cognitoで管理されます。
パスワード Amazon Cognitoによりハッシュ化して管理します。当サービスが平文のパスワードを保存することはありません。
表示名 アプリ内でのあなたの名前（15文字以内）。パートナーにも表示されます。

プロフィール情報

情報 説明
アバター画像 任意で設定できます。Amazon S3に保存され、あなた自身とペアリングしたパートナーのみがアクセスできます。画像への直接アクセスは許可されておらず、署名付きURL（5分間有効）経由でのみ取得可能です。

きろく・活動情報

ユーザーが入力した以下の情報をAmazon DynamoDBに保存します。

  • きろくの各フィールド（事実・気持ち・原因・自分の解決策・パートナーへのお願い）
  • コメント・リアクション絵文字
  • 解決フラグの状態
  • きろくの共有日時・共有状態

これらのデータは作成者本人とペアリングしたパートナー以外はアクセスできません。第三者（当サービスの運営者を含む）が内容を閲覧することはありません。

通知情報

情報 説明
デバイストークン（APNs） プッシュ通知を有効にした場合のみ収集します。パートナーからのきろく共有・コメント・リアクション・解決フラグ時の通知に使用します。通知をオフにするとトークンは削除されます。

サブスクリプション情報

情報 説明
プラン情報 現在のプラン（free / premium）と有効期限。RevenueCatのWebhook経由で自動更新されます。

プレミアムプランの課金管理にRevenueCatを使用しています。購入処理時に、RevenueCatへCognito user IDおよびApp Storeのレシート情報が送信されます。

2. 情報の利用目的

収集した情報は以下の目的にのみ使用します。

情報 利用目的
メールアドレス・パスワード 認証・ログイン管理、メール確認
表示名・アバター画像 アプリ内での識別表示（自分とパートナーの画面）
きろく・コメント・リアクション サービス機能（作成・共有・仲直りフロー）の提供
デバイストークン プッシュ通知の送信
サブスクリプション情報 プレミアムプランの機能制限管理

当サービスは、広告目的・マーケティング分析・行動追跡にユーザーデータを使用しません。アクセス解析ツール（Google Analyticsなど）は導入していません。

3. 第三者へのデータ提供

当サービスは以下のサービスプロバイダーを利用しています。これら以外の第三者にユーザーの個人情報を販売・提供することはありません。

Amazon Web Services（AWS）

当サービスのデータはAWSのインフラ上（東京リージョン）で処理・保存されます。AWSはデータプロセッサーとして機能し、当サービスの指示に従ってのみデータを処理します。

対象サービス：Amazon Cognito（認証）、Amazon DynamoDB（データ保存）、Amazon S3（アバター画像）、AWS Lambda（バックエンド処理）、Amazon SNS（プッシュ通知配信）

AWSプライバシーポリシー →

RevenueCat

プレミアムプランのサブスクリプション管理に使用しています。購入・更新・解約時にCognito user IDとApp Storeのレシート情報がRevenueCatに送信されます。

RevenueCatプライバシーポリシー →

Apple（APNs・App Store）

プッシュ通知：通知の配信にApple Push Notification Service（APNs）を利用しています。通知送信時にデバイストークンがAppleのサービスを通じて処理されます。

App Store決済：プレミアムプランの課金はApple App Storeを通じて処理されます。決済情報はAppleが管理し、当サービスはカード番号等の決済情報を受け取りません。

Appleプライバシーポリシー →

なお、法令に基づく開示要求がある場合は、必要最小限の情報を開示することがあります。

4. データへのアクセス制御

データへのアクセスはAWS Amplify Gen 2の認可機能によって以下のように制限されています。

  • きろく：作成者本人とペアリングしたパートナーのみが閲覧可能
  • プロフィール情報：本人とペアリングしたパートナーのみが閲覧可能
  • コメント・リアクション：作成者本人とペアリングしたパートナーのみが閲覧可能
  • アバター画像：署名付きURLでのみアクセス可能。URLは5分間有効。本人と双方向にペアリング確認済みのパートナーのみ取得可能

5. データの保持と削除

アカウント削除

アプリ内の設定画面からアカウントを削除すると、以下のデータがすべて即時に削除されます。

  • メールアドレス・アカウント情報（Amazon Cognito）
  • プロフィール情報（表示名・ペアリング情報）
  • すべてのきろく・コメント・リアクション（Amazon DynamoDB）
  • アバター画像ファイル（Amazon S3）
  • 招待コード・ペアリング情報

削除後のデータ復元はできません。また、アカウント削除時にはパートナーとのペアリングも自動的に解除されます。

ペアリング解除

パートナーとのペアリングを解除すると、以下のデータが削除されます。

  • 相互のコメント・リアクション・解決フラグ
  • きろくの共有状態（きろく本文自体は削除されません）

通常利用中のデータ保持

アカウントを保持している間は、作成したきろく・プロフィール情報・コメント等を保持し続けます。特定の保持期限は設けていません。

6. ユーザーの権利

ユーザーはいつでも以下の操作が可能です。

権利 操作方法
データへのアクセス アプリ内でいつでも自分のきろく・プロフィールを確認できます
データの修正 表示名・アバター画像・きろく内容（未共有のもの）はアプリ内から変更できます
データの削除 設定画面のアカウント削除機能により、全データを即時削除できます
プッシュ通知の停止 設定画面でいつでもオフにできます
ペアリングの解除 設定画面からいつでもパートナーとのペアリングを解除できます

7. 未成年者について

当サービスは恋人・夫婦間での感情共有を目的としており、16歳以上の方を対象としています。16歳未満の方のご利用はお断りしております。保護者の方が16歳未満のお子様によるアカウント登録を発見した場合は、お問い合わせ先までご連絡ください。

8. クッキー・ローカルストレージ

当サービスはサービス提供に必要な以下の情報をデバイスに保存します。

  • 認証トークン：Amazon Cognitoのセッション管理用（ログイン状態の維持）
  • 言語設定：アプリの表示言語（日本語/英語）
  • 通知設定：プッシュ通知のオン/オフ状態

広告目的のクッキーや第三者トラッキングは使用していません。

9. 本ポリシーの変更

本ポリシーは予告なく変更することがあります。変更した場合は本ページを更新します。変更の際に個別の通知は行いません。変更後も継続して当サービスを利用した場合、変更後のポリシーに同意したものとみなします。

お問い合わせ

個人情報の取り扱いに関するご質問・削除依頼等は、App Store のレビューよりお願いします。
